# Actualizacion de la app WeatherApp 
  
## Instalación  
npm install  

## Ejecución  
npm start

## Variables de entorno  
``VITE_API_KEY``  

**Nota:** Para obtener la API_KEY, es necesario registrarse en la página de [OpenWeather](https://openweathermap.org/)  

## Tecnologías utilizadas
- React  
- Typescript  
- Vite  
- React Router Dom  
- Material UI  
- Axios   
- React Icons  
- Formik
